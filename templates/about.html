About this site

This web application was created to develop my MVC and Python skills as well as perhaps someday providing a place where students can ask questions to their peer group and get jargon-free answers on a varied subject matter.

It demonstrates the following features:

  • Development with the Django MVC framework.
  • Using ORM to store users posts and comments.
  • Persistent user login and user registration.
  • Responsive, mobile friendly design with CSS.
  • Dynamic content (search, pagination and explicit content filtering) with AJAX.
  • Version Control with Git. This project can be viewed on my Github page
  • Deployment to Heroku.
